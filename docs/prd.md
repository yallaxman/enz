# PRD: демо расчета КП по спецификации

## Цель

Показать заказчику понятный рабочий пример: из спецификации и прайса получается редактируемая ведомость с ценами и выгрузкой в Excel.

## Пользовательский сценарий

1. Пользователь загружает или выбирает пример спецификации.
2. Пользователь загружает или выбирает пример прайса.
3. Система приводит наименования к единому виду.
4. Система подбирает подходящие позиции из прайса.
5. Пользователь проверяет спорные строки.
6. Пользователь выгружает результат в Excel.

## Что входит в демо

- Моковая спецификация по инженерным сетям.
- Моковый прайс поставщика.
- Загрузка пользовательских CSV/XLSX-файлов с типовыми русскими колонками.
- Автоматическое сопоставление строк.
- Уровень уверенности по каждой позиции.
- Ручное редактирование таблицы.
- Сохранение расчета.
- Выгрузка Excel.
- Черновик коммерческого предложения на основе сохраненного отчета.

## Что не входит в демо

- Гарантированный разбор любых исходных документов.
- Подключение реальных поставщиков.
- Роли пользователей.
- Интеграция с бухгалтерией, CRM или ERP.
- Полная автоматизация без проверки человеком.

## Критерий успешного показа

Клиент должен увидеть, что рутинный процесс можно сократить до понятного потока:

`спецификация -> подбор цен -> проверка -> Excel`.
