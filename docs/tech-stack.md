# Технологический подход

## Для демо

- Локальное веб-приложение без сложной инфраструктуры.
- Моковые данные спецификации и прайса.
- Табличный интерфейс с ручным редактированием.
- Экспорт результата в Excel.
- Сохранение расчетов для истории показа.

## Для рабочего MVP

- Веб-интерфейс с личным кабинетом.
- Загрузка исходных документов и таблиц.
- Извлечение спецификаций в редактируемый вид.
- Справочник поставщиков и прайс-листов.
- Сопоставление позиций с оценкой уверенности.
- Ручная проверка перед финальной выгрузкой.
- Экспорт в Excel по шаблону клиента.

## Почему так

На первом показе важно не обсуждать внутреннюю инфраструктуру, а показать ценность:

- меньше ручного переноса строк;
- быстрее поиск цен;
- видны спорные места;
- результат можно отдать дальше в работу.

Технические детали хранения данных, хостинга и интеграций лучше обсуждать после подтверждения интереса и бюджета.
