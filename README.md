# Демо расчета КП по спецификации

Локальный пример для показа заказчику: загружаем спецификацию и прайс, получаем сопоставленную ведомость, проверяем спорные строки и выгружаем Excel.

## Запуск

```bash
python3 app/main.py
```

Открыть:

```text
http://127.0.0.1:8000
```

## Сценарий показа

1. Нажать `Большой тест` в блоке спецификации или выбрать свой `CSV/XLSX`.
2. Нажать `Большой тест` в блоке прайса или выбрать свой `CSV/XLSX`.
3. При желании нажать `Нормализовать позиции`.
4. Нажать `Сопоставить`.
5. Показать найденные позиции, цены, суммы и строки “нужно проверить”.
6. Отредактировать одну строку вручную, чтобы показать контроль человеком.
7. Нажать `Сохранить`.
8. В блоке `Сохраненные отчеты` скачать результат в `Excel`, `CSV` или `КП DOCX`.

## Что важно проговорить

Это демонстрация процесса на моковых данных. Реальный проект дальше подключается к фактическим прайсам, исходным документам и шаблону КП заказчика.

## Тестовые файлы

- `samples/test_spec_small.csv` — спецификация на 5 строк.
- `samples/test_price_small.csv` — прайс на 5 строк.
- `samples/test_spec_full.csv` — расширенная спецификация на 20 строк с русскими заголовками.
- `samples/test_price_full.csv` — расширенный прайс на 22 строки с русскими заголовками.
- `samples/test_spec_full.xlsx` — та же расширенная спецификация в Excel-формате.
- `samples/test_price_full.xlsx` — тот же расширенный прайс в Excel-формате.

## Какие колонки распознаются

Для спецификации: `Наименование`, `Ед. изм.`, `Кол-во`, `Поз.`, `Раздел`, `Примечание`.

Для прайса: `Артикул`, `Наименование`, `Ед. изм.`, `Цена`, `Производитель`, `Срок поставки`.

Можно загружать CSV с `,` или `;`, а также простые XLSX-файлы с таблицей на первом листе.
